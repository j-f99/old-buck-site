2. Visual Aesthetic ("The Executive Vibe")
	•	Style: High-end, "Heritage" luxury. Think vintage yacht club or executive lounge.
	•	Backgrounds: Use the Midnight Navy for deep sections and the Soft Cream for high-readability text blocks.
	•	Typography: Elegant Serif for headlines (Classic/Timeless); clean Sans-Serif for body copy.
	•	Animations: Focus on "Elevator" transitions—vertical movements and subtle gold-bordered button hovers.
3. Technical Stack
	•	Framework: Next.js (React) + Tailwind CSS + Framer Motion.
	•	Deployment: Static Export for Nixihost.
